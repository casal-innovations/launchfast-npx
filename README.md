# @launchfasthq/create

Scaffold a new [LaunchFast](https://launchfast.pro) project.

## Usage

```bash
npx @launchfasthq/create
```

This will run the interactive CLI to set up your project.

## What this package does

This is a lightweight wrapper around [@launchfasthq/cli](https://www.npmjs.com/package/@launchfasthq/cli). It exists so you can run `npx @launchfasthq/create` without installing anything globally.

## Requirements

- Node.js 18 or higher
